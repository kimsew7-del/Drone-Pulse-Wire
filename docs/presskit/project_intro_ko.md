# Briefwave 프로젝트 소개

## 한 줄 소개

Briefwave는 드론과 AI 분야의 글로벌 뉴스, 논문, 리포트를 자동으로 수집하고, 사용자의 관심 키워드에 맞춰 개인화된 인사이트 피드로 재구성하는 뉴스 인텔리전스 플랫폼입니다.

## 프로젝트 개요

드론과 AI 시장은 기술 변화 속도가 빠르고, 정보가 여러 국가·언어·매체에 분산되어 있습니다. Briefwave는 이 문제를 해결하기 위해 다양한 뉴스 소스와 학술 데이터베이스를 연결하고, 수집된 정보를 하나의 흐름으로 정리합니다. 사용자는 관심 키워드와 카테고리, 지역, 언어 조건을 설정해 자신만의 맞춤형 정보 피드를 만들 수 있습니다.

단순히 뉴스를 모아 보여주는 수준을 넘어, Briefwave는 중복 기사 정리, 카테고리 자동 분류, 번역, 트렌드 분석, 구독 기반 수집까지 연결해 "읽을 거리"를 "의사결정용 정보"로 전환하는 데 초점을 둡니다.

## 해결하려는 문제

- 드론·AI 관련 정보가 글로벌 매체, 연구기관, 검색형 API에 흩어져 있어 추적 비용이 큼
- 같은 이슈가 여러 매체에 중복 보도되어 핵심 변화만 파악하기 어려움
- 해외 기사와 연구 자료는 언어 장벽 때문에 빠르게 소비하기 어려움
- 사용자가 원하는 분야만 따로 모아보는 개인화 뉴스 환경이 부족함

## Briefwave의 핵심 가치

- 자동 수집: 뉴스, 연구 자료, 리포트를 주기적으로 수집
- 개인화 피드: 키워드 구독 기반으로 사용자별 맞춤 뉴스 구성
- AI 보조 분석: 요약, 카테고리화, 신호 추출, 트렌드 해석 지원
- 번역 지원: 해외 기사도 한국어 중심으로 빠르게 파악 가능
- 운영 가시성: 소스 상태, 수집 현황, 피드 통계를 함께 관리 가능

## 주요 기능

### 1. 키워드 구독 기반 맞춤형 수집

사용자는 관심 주제에 맞춰 구독을 생성하고, 키워드·카테고리·지역·언어·수집 기간을 설정할 수 있습니다. 각 구독은 독립적으로 관리되며 필요할 때 직접 크롤링을 실행할 수 있습니다.

### 2. 드론·AI 특화 뉴스 인텔리전스

Briefwave는 드론, UAM, AI, 로보틱스, 연구 동향처럼 빠르게 바뀌는 분야를 추적하기 위한 구조를 갖추고 있습니다. 일반적인 뉴스 리더보다 기술 변화와 산업 신호를 빠르게 포착하는 데 더 적합합니다.

### 3. 글로벌 소스 통합

RSS, 검색형 뉴스 API, 학술 데이터베이스 등 여러 형태의 소스를 연결해 단일 피드로 통합합니다. 이를 통해 언론 기사와 연구 자료를 한 화면에서 함께 볼 수 있습니다.

### 4. 번역 및 요약 보조

해외 기사도 한국어 번역과 요약 정보를 바탕으로 빠르게 이해할 수 있습니다. 글로벌 기술 동향을 국내 사용자 관점에서 소비하기 쉽게 만드는 기능입니다.

### 5. 분석과 모니터링

피드에는 기사 목록만 있는 것이 아니라, 신호 데이터, 트렌드 스냅샷, 카테고리 분석, 소스 상태와 같은 운영형 정보도 포함됩니다. 서비스 운영자와 사용자 모두 정보 흐름을 더 입체적으로 볼 수 있습니다.

## 기대 활용 시나리오

- 드론 산업 동향을 매일 추적해야 하는 스타트업 팀
- AI와 UAM 시장 변화를 모니터링하는 전략/기획 담당자
- 해외 기술 기사와 연구 동향을 빠르게 훑어야 하는 연구자
- 특정 키워드 기반으로 개인 뉴스 워크플로를 만들고 싶은 사용자

## 기술적 특징

- Frontend: Next.js 기반 UI
- Backend: FastAPI 기반 API 서버
- Data Layer: SQLite 기반 저장 및 피드 관리
- Core Capability: 크롤링, 구독 관리, 번역, 카테고리 분석, 개인화 피드 제공

## 소개 문구 예시

### 짧은 소개

### 발표/제안서용 소개

Briefwave는 드론·AI 산업에서 흩어져 있는 뉴스, 논문, 리포트를 하나로 통합하고, 사용자가 설정한 키워드와 조건에 따라 개인 맞춤형 인사이트 피드를 제공하는 플랫폼입니다. 자동 수집, 번역, 분류, 트렌드 분석 기능을 통해 정보 탐색 시간을 줄이고 더 빠른 판단을 지원합니다.

### 마케팅용 소개

복잡한 정보 탐색은 줄이고, 중요한 변화만 빠르게 잡아내는 드론·AI 뉴스 인텔리전스. Briefwave는 글로벌 소스를 자동으로 읽고, 당신의 관심사에 맞는 신호만 선별해 전달합니다.
